## Usage

```jsx
import {
  SegmentedControl,
  SegmentedControlItem,
} from "@/registry/ui/segmented-control"
```

```jsx
<SegmentedControl defaultValue="Option 1">
  <SegmentedControlItem value="Option 1">Option 1</SegmentedControlItem>
  <SegmentedControlItem value="Option 2">Option 2</SegmentedControlItem>
</SegmentedControl>
```

### Accessibility

Adheres to the [Radio Group WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/radio/). The `SegmentedControl` component uses the `role="radio"` attribute to indicate that the component is a linear set of buttons represented as a single element. The `aria-checked` attribute is used to indicate whether the component is selected.

### Keyboard Navigation

| Key                   | Action                                                                                                                                                                                                                                                                      |
| --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <kbd>ArrowRight</kbd> | Move focus to the next `SegmentedControlItem` in the group, uncheck the previously focused `SegmentedControlItem`, and check the newly focused `SegmentedControlItem`. If focus is on the last `SegmentedControlItem`, focus moves to the first `SegmentedControlItem`.     |
| <kbd>ArrowLeft</kbd>  | Move focus to the previous `SegmentedControlItem` in the group, uncheck the previously focused `SegmentedControlItem`, and check the newly focused `SegmentedControlItem`. If focus is on the first `SegmentedControlItem`, focus moves to the last `SegmentedControlItem`. |
